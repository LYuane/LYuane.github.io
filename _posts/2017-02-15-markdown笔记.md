> 随便整理的自用md

# 0x01 概述

markdown语法使创作文档更致力于内容而非排版
HTML 是一种 发布 格式; Markdown 是一种 创作 格式. 因此, Markdown 处理的都是纯文本.
tips：< &

# 0x02 块级元素

## 标题
md语法

    #  一级标题
    ##  二级标题
    ### 三级标题
    #### 四级标题
    ##### 五级标题 
    ###### 六级标题

对应md样式

![](img/2017-02-15-markdown/md-title.png)

## 块引用

    >this is a blockquote
![](img/2017-02-15-markdown/md-blockquote1.png)

块引用可以嵌套
    >this is the first level of quoting
    >>this is nested blockquote
    >>> this is final blockquote

![](img/2017-02-15-markdown/md-blockquote2.png)

同时块引用里面还可以包含markdown元素，例如标题，列表，代码块等

## 列表
支持有序列表和无序列表(使用markdownPad有快捷键)

无序列表可以使用* + -

这些符号都是等价的

对于最终生成HTML没有影响  

官方文档示例
![](img/2017-02-15-markdown/md-list1.png)


    * 1
    * 2
    * 3
    
    
    + 1
    + 2
    + 3
    
    - 1
    - 2
    - 3

![](img/2017-02-15-markdown/md-list2.png)

## 代码块
插入代码块只需要将每一行都缩进 4 个空格或者 1 个水平制表符

	this is a code block

![](img/2017-02-15-markdown/md-code.png)

## 水平线

如果一行中只有三个以上的连字符, 星号, 或者下划线都会生成一条水平线(星号和连字符之间的空格也是允许的)

    * * *
    
    ***
    
    *****
    
    - - -
    
    ---------------------------------------

![](img/2017-02-15-markdown/md-line.png)

# 0x03 内联元素

## 链接
格式：[文本]（URL）

    this is [an example](http://example.com/)

## 代码
要输出一个代码片段, 需要使用重音符号 (`). 不同于预格式的代码块, 代码片段只是在普通段落中标识出代码

eg：Use the `printf()` function.

## 图片

![Alt text](/path/to/img.jpg)

- 一个感叹号: !;
- 紧跟着一对方括号, 包含了图片的 alt 属性;
- 紧跟着一对圆括号, 包含了图片的 URL 或者路径, 以及一个可选的用单引号或双引号包裹的 title 属性.

Markdown 没有语法指定图片尺寸; 如果需要指定图片尺寸, 可以使用 HTML <img> 标签

# 0x04 其他

## 自动链接
Markdown 支持一种 "自动" 创建 URL 和 email 地址链接的简短形式: 只需用尖括号包围 URL 或 email 地址即可

## 反斜杠转义

使用反斜杠转义 Markdown 语法符号为字面量

例如, 如果你想用星号包围一个单词 (而不是 HTML 的 <em\> 标签), 你可以在星号前面加反斜杠

    * literal asterisks*
    
    \*literal asterisks\*

![](img/2017-02-15-markdown/md-escape.png)

Markdown 为下面字符提供反斜杠转义:

    \   backslash
    `   backtick
    *   asterisk
    _   underscore
    {}  curly braces
    []  square brackets
    ()  parentheses
    #   hash mark
    +   plus sign
    -   minus sign (hyphen)
    .   dot
    !   exclamation mark
