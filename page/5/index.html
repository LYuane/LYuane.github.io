docker安装

发表于 2018-03-05

0x00 docker版本

CE 社区版（免费）
EE 企业版，安全，付费

安装环境为ubuntu16.04

tips：16.04+的版本无需设置内核模块

0x01 APT安装

更新apt软件包缓存

安装apt

$ sudo apt-get install \
apt-transport-https \
ca-certificates \
curl \
software-properties-common

添加软件源的 GPG 密钥，为了确认所下载软件包的合法性

然后，我们需要向 source.list 中添加 Docker 软件源

tips：鉴于国内网络问题，强烈建议使用国内源

0x02 安装docker CE

更新 apt 软件包缓存，并安装 docker-ce：

$ sudo apt-get update

$ sudo apt-get install docker-ce

启动 Docker CE

本机为ubuntu16.04

$ sudo systemctl enable docker
$ sudo systemctl start docker

Ubuntu 14.04 使用以下命令启动：

$ sudo service docker start

建立 docker 用户组

默认情况下，docker 命令会使用 Unix socket 与 Docker 引擎通讯。而只有 root 用户和 docker 组的用户才可以访问 Docker 引擎的 Unix socket。出于安全考虑，一般 Linux 系统上不会直接使用 root 用户。因此，更好地做法是将需要使用 docker 的用户加入 docker 用户组。

建立 docker 组：

$ sudo groupadd docker

将当前用户加入 docker 组：

$ sudo usermod -aG docker $USER

退出当前终端并重新登录，进行如下测试。

0x03 遇到的问题

当我使用当前用户启动docker时，显示如下信息

该错误消息告诉您当前用户无法访问docker引擎，因为您缺少访问unix套接字以与引擎通信的权限

原因是我将用户加入用户组后没有重启！！！重启下就好了

如果大家遇到这种报错可以参考以下两种方案

解决方案：

  • 每次使用sudo以root身份运行（不推荐）

  • 将当前用户加入用户组，然后重启，重启！

markdown语法整理

发表于 2018-02-15

0x01 概述

markdown语法使创作文档更致力于内容而非排版

HTML 是一种 发布 格式; Markdown 是一种 创作 格式.

因此, Markdown 处理的都是纯文本.

tips：< &

0x02 块级元素

标题

md语法

#  一级标题
##  二级标题
### 三级标题
#### 四级标题
##### 五级标题 
###### 六级标题

对应md样式

块引用

>this is a blockquote

块引用可以嵌套

>this is the first level of quoting
>>this is nested blockquote
>>> this is final blockquote

同时块引用里面还可以包含markdown元素，例如标题，列表，代码块等

列表

支持有序列表和无序列表(使用markdownPad有快捷键)

无序列表可以使用* + -

这些符号都是等价的

对于最终生成HTML没有影响

官方文档示例

示例

代码块

插入代码块只需要将每一行都缩进 4 个空格或者 1 个水平制表符

this is a code block

水平线

如果一行中只有三个以上的连字符, 星号, 或者下划线都会生成一条水平线(星号和连字符之间的空格也是允许的)

* * *

***

*****

- - -

---------------------------------------

0x03 内联元素

链接

格式：[文本]（URL）

this is [an example](http://example.com/)

代码

要输出一个代码片段, 需要使用重音符号 (`). 不同于预格式的代码块, 代码片段只是在普通段落中标识出代码

eg：

Use the \`printf()` function.

Use the printf() function.

图片

 ![Alt text](/path/to/img.jpg)
  • 一个感叹号: !;
  • 紧跟着一对方括号, 包含了图片的 alt 属性;
  • 紧跟着一对圆括号, 包含了图片的 URL 或者路径, 以及一个可选的用单引号或双引号包裹的 title 属性.

Markdown 没有语法指定图片尺寸; 如果需要指定图片尺寸, 可以使用 HTML <img> 标签

0x04 其他

自动链接

Markdown 支持一种 “自动” 创建 URL 和 email 地址链接的简短形式: 只需用尖括号包围 URL 或 email 地址即可

反斜杠转义

使用反斜杠转义 Markdown 语法符号为字面量

例如, 如果你想用星号包围一个单词 (而不是 HTML 的 <em> 标签), 你可以在星号前面加反斜杠

* literal asterisks*

\*literal asterisks\*

Markdown 为下面字符提供反斜杠转义:

\   backslash
`   backtick
*   asterisk
_   underscore
{}  curly braces
[]  square brackets
()  parentheses
#   hash mark
+   plus sign
-   minus sign (hyphen)
.   dot
!   exclamation mark
